Use and Examples
================

The examples show the basic usage of **pyIMD** to calculate the mass

The pyIMD UI can be started from source with from within the `pyIMD/pyIMD` folder:

.. code-block:: console

     python3 main.py

An example script can be started from within the `pyIMD` folder:

.. code-block:: console

    python3 pyIMD/examples/example_usage_script.py

Alternatively download the compiled executable for your platform as described under installation.
